# Support

Umami Compass is community-maintained and provided without an SLA.

- Read the [README](README.md), especially configuration, client setup, compatibility, and troubleshooting.
- Use [GitHub Discussions](https://github.com/webcredo/umami-compass/discussions) for setup questions, use cases, and design conversations.
- Use the issue templates for reproducible bugs and scoped feature proposals.
- Report vulnerabilities only through GitHub private vulnerability reporting as described in [SECURITY.md](SECURITY.md).

Include the Umami Compass version, Umami version/deployment, Node version, MCP client, selected toolsets, and a minimal reproduction with synthetic identifiers. Never include credentials or real visitor/session/replay data.
